# 一、主页

- 顶部：中间显示“微信”，右侧有搜索（放大镱图标，跳转搜索页）、更多（加号图标，点击展开菜单）

    - 更多按钮点击展开后有：发起群聊、添加朋友、扫一扫、收付款

- 中间：显示单聊、群聊会话列表，按时间排序，可上下翻阅，点击可进入对应聊天页，长按可展开菜单（取消置顶、删除该聊天）

- 底部：第一个“微信”按钮，点击可返回主页

# 二、搜索页面

- 顶部：左侧返回按钮（回主页），中间搜索框（可搜联系人、群聊、聊天记录），右侧更多按钮（查看/清空搜索历史）

- 中间：显示搜索结果，点击可进入对应聊天页或定位消息

# 三、个人聊天页（单聊页）

- 顶部：左侧返回按钮（回主页），中间显示对方头像/昵称，、更多按钮（聊天相关操作：查找聊天记录、消息免打扰、置顶、清空聊天记录、投诉）

- 中间：显示双方聊天消息，可上下滑动查看历史记录

- 底部：最左侧语音图标，中间输入框（发文字），右侧有表情图标（发表情）、加号图标（发图片、文件、视频通话（点击弹出选择视频还是音频通话）、红包等），当输入框内有文字时，右侧变成发送按钮

# 四、群聊页

- 顶部：左侧返回按钮（回主页），中间显示群头像/群名，右侧更多按钮（群设置、群管理等）

- 中间：显示群内所有成员聊天消息，可上下滑动查看历史记录

- 底部：与个人聊天页一致，可发文字、表情、图片等，支持@群成员
